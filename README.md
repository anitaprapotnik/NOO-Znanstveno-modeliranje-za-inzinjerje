# NOO-Znanstveno-modeliranje-za-inzinjerje

Lepo pozdravljeni v kratkem izobraževanju Znanstveno modeliranje za inžinerje, ki je nastal v sklopu projekta NOO. 
Prosim da si pred začetkom natančno preberete navodila (v nadaljevanju). 

Kratko izobraževanje NOO Znanstveno modeliranje vas bo popeljalo skozi osnovne ideje nevronskih mrež in njihovo implementacijo v programskem jeziku Python. Po končanem modulu boste spoznali:

 - zakaj se uporabljajo računalniško generirana naključna števila,

 - kako v vašem programu uporabiti računalniško generirana naključna števila, porazdeljena po osnovnih statističnih distribucijah,

 - terminologijo in izraze, povezane z nevronskimi mrežami,

 - kako v Pythonu sestaviti in natrenirati preprosto (gosto) nevronsko mrežo za iskanje vzorcev v podatkih,

 - kako v Pythonu sestaviti in natrenirati konvolucijsko nevronsko mrežo za prepoznavanje objektov na slikah.

## Pogoji

Če želite uspešno slediti kratkemu izobraževanju, potrebujete naslednje znanje in programsko opremo:

***Programska oprema***

 - Jupyter Notebook ali JupyterLab
 - osnovni program Python
 - Python knjižnice:
   - numpy
   - matplotlib
   - scipy
   - pandas
   - tensorflow
   - tensorflow-datasets
   - keras

Najlažje namestite vso potrebno programsko opremo tako, da namestite paket Anaconda, ki vključuje:

 - Jupyter Notebook in JupyterLab,
 - Python z najpogosteje uporabljenimi knjižnicami (numpy, scipy, matplotlib, pandas).

Navodila za namestitev najdete na (brezplačna namestitev):

https://www.anaconda.com/download

V tem primeru je treba dodatno namestiti še knjižnice keras, tensorflow in tensorflow-datasets. Knjižnice lahko namestite tako, da odprete priloženo Jupyter Notebook datoteko (instalacija) in zaženete program v njej. 

 ***Znanje***

## Sestava kratkega izobraževanja
Izobraževanje je sestavljeno iz treh sklopov:

 1. Naključna števila
 2. Goste nevronske mreže (ANN)
 3. Konvolucijske nevronske mreže (CNN)
    
Vsak izmed treh sklopov je organiziran na naslednji način:

 - lekcija s teoretično osnovo,
 - Jupyter Notebook datoteka ali datoteke s primeri
 - dodatno, z UI generirano gradivo,
 - morebitno dodatno gradivo.
